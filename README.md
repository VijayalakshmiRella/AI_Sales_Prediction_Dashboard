# AI Sales Prediction Dashboard

## Project Description

The AI Sales Prediction Dashboard is a Machine Learning application that predicts future sales using historical sales data. It provides an interactive Streamlit dashboard with charts, predictions, and business insights.

## Objectives

- Predict future sales
- Analyze sales data
- Visualize trends
- Support business decision-making

## Features

- Data preprocessing
- Feature engineering
- Machine learning model
- Interactive dashboard
- Sales prediction
- Charts and visualizations
- Prediction history

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Plotly
- Joblib
- Matplotlib
